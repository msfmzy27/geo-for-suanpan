# geo-for-suanpan
a geo generator for suanpanmall.com
## auto-feed is the directory with a test for auto-feeding article to CSDN
## text is the directory with methods of processing data and the data itself
## demo shows how to generate a rag-friendly text with only a few words. demo_withdify shows how to generate through a dify workflow.
## eval shows a method to measure the effectiveness of GEO which is from the original paper.
## general_optimized is a try to find general prompt for llm.
## test1, optimized1 and optimized2 try to make a html become a GEO text according to the functions from original GEO.
