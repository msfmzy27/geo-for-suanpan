# geo-for-suanpan
a geo generator for suanpanmall.com
## auto-feed is the directory with a test for auto-feeding article to CSDN
## text is the directory with methods of processing data and the data itself
## auto_g gives a way to generate batches of GEO texts automatically
## demo.py shows how to generate a rag-friendly text with only a few words. demo_withdify.py shows how to generate through a dify workflow.
## eval.py shows a method to measure the effectiveness of GEO which is from the original paper.
## general_optimized.py is a try to find general prompt for llm.
## test1.py, optimized1.py and optimized2.py try to make a html become a GEO text according to the functions from original GEO.
